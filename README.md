# GitHub練習用プロジェクト

このプロジェクトは、GitHubの基本的な操作を練習するためのシンプルな電卓アプリケーションです。

## 機能

- 足し算
- 引き算
- 掛け算
- 割り算

## 使い方

1. リポジトリをクローンします
2. `calculator.py`を実行します：
   ```bash
   python calculator.py
   ```
3. 表示されるメニューから操作を選択し、数字を入力します
